🤖 Gemini-Powered Data Intelligence Hub
Overview
This application is a conversational data analyst built to bridge the gap between raw datasets and actionable insights. By combining Python with Google Gemini 1.5 Flash, users can upload CSV files and ask complex questions about their data in plain English.
🚀 Key Features
• Automated Data Summarization: Uses Pandas to instantly generate descriptive statistics and handle data cleaning.
• Natural Language Querying: Integrated with Google AI Studio to interpret user questions and provide context-aware answers based on the uploaded data.
• Interactive Dashboard: A streamlined, professional UI built with Streamlit for a seamless user experience.
• Secure API Integration: Demonstrates best practices in handling environment variables and API secrets.
🛠️ Tech Stack
• Language: Python 3.10+
• AI Model: Google Gemini 1.5 Flash (via google-generativeai)
• Data Processing: Pandas, NumPy
• Frontend: Streamlit
• Development Environment: Replit
📖 How to Use
1. Upload: Drop a .csv file into the uploader (e.g., sales data, inventory, or Chicago tech salaries).
2. Analyze: View the automated summary generated by the Python backend.
3. Chat: Ask questions like "What was the highest-grossing category?" or "Summarize the trends for the last quarter."
